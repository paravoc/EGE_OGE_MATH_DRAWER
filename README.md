# Математика рисовалка

Интерактивная веб-доска для уроков и объяснения математических тем. Текущая версия реализована как локальный MVP в одном HTML-файле и позволяет рисовать на бесконечном полотне, добавлять текст, заметки, фигуры, графики, изображения и формулы.

## Что умеет приложение

- рисование кистью по холсту;
- стирание штрихов и удаление целых объектов;
- добавление текста и заметок;
- вставка 2D-фигур, 3D-фигур и базовых графиков функций;
- автоматическая подпись вершин у фигур;
- поиск по справочнику формул и вставка формулы на холст;
- загрузка изображений с диска;
- вставка текста и изображений через буфер обмена;
- выделение, групповое выделение рамкой, перемещение и масштабирование объектов;
- дублирование и удаление выделенных элементов;
- масштабирование холста и панорамирование;
- экспорт результата в PNG;
- автоматическое сохранение состояния между сеансами через `localStorage`.

## Текущее состояние проекта

Сейчас проект представляет собой одностраничный MVP без сборки и без серверной части:

- основной файл приложения: `v 1 0 0.html`;
- интерфейс, стили и логика находятся в одном файле;
- внешние зависимости подключаются через CDN;
- данные сохраняются локально в браузере.

Это удобно для быстрой демонстрации идеи, но для дальнейшего развития проект стоит разделить на HTML, CSS, JS и данные формул.

## Стек и зависимости

- HTML5
- CSS3
- JavaScript
- Canvas API
- [MathJax](https://www.mathjax.org/) для отображения математических формул
- [html-to-image](https://github.com/bubkoo/html-to-image) для преобразования формул в изображение перед вставкой на доску

Важно: для корректной работы формул и части экспорта нужен доступ к интернету, так как библиотеки подключаются с CDN.

## Основные сценарии использования

### 1. Быстрое объяснение темы

Можно рисовать от руки, добавлять подписи, вставлять геометрические фигуры и показывать базовые графики функций на одном полотне.

### 2. Подготовка математических схем

Приложение подходит для построения простых наглядных схем, подписей к чертежам и иллюстрации формул.

### 3. Экспорт результата

Готовую доску можно сохранить в PNG и использовать в презентации, отчёте или учебных материалах.

## Горячие клавиши и управление

### Инструменты

- `V` - выделение
- `B` - кисть
- `E` - ластик штрихов
- `O` - ластик объектов
- `T` - текст
- `Alt + T` - быстрый переход к тексту
- `N` - заметка

### Редактирование

- `Ctrl + Z` - отмена
- `Ctrl + Shift + Z` или `Ctrl + Y` - повтор
- `Delete` / `Backspace` - удалить выделенное
- `Ctrl + A` - выделить всё
- `Ctrl + C` - скопировать текст выбранного текстового объекта
- `Ctrl + D` - дублировать выделение

### Навигация по холсту

- `Колёсико мыши` - масштаб
- `Ctrl + колёсико` - изменение размера кисти
- `Space` - временное панорамирование
- `Правая кнопка мыши` - панорамирование
- `Ctrl + ЛКМ` - быстрый режим выделения/перемещения

### Работа с объектами

- фигуры можно добавлять кнопкой `Добавить`
- фигуры можно перетаскивать из меню прямо на холст
- изображения можно загружать через кнопку `Загрузить картинку`
- изображения и текст можно вставлять через буфер обмена
- формулы вставляются кликом по элементу в панели `Формулы`

## Как устроен MVP

Приложение работает как редактор сцены поверх `canvas`:

- все объекты доски хранятся в памяти браузера;
- после изменений состояние сериализуется в `localStorage`;
- при повторном открытии страницы состояние восстанавливается;
- формулы сначала рендерятся через MathJax, затем превращаются в изображение и вставляются на холст как объект;
- экспорт в PNG выполняется через отдельный вспомогательный canvas.

## Ограничения текущей версии

- весь код находится в одном HTML-файле;
- отсутствует модульная структура;
- нет серверной части и совместного доступа;
- нет сохранения в собственный формат проекта;
- сохранение работает только в пределах конкретного браузера и профиля;
- формулы и часть возможностей зависят от внешних CDN;
- нет автоматических тестов.

## Что можно улучшить дальше

### Ближайшие практичные шаги

- совместная работа в реальном времени.
